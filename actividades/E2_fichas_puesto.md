## Objetivo de Aprendizaje

Aplicar la metodología WorldatWork para elaborar descripciones de puesto estandarizadas que servirán como base para la evaluación por puntos y el diseño del tabulador salarial.

## Contexto

Geotest cuenta con 33 puestos distribuidos en 5 niveles jerárquicos. Para diseñar un sistema de compensaciones equitativo, necesitamos documentar de manera sistemática las responsabilidades, requisitos y condiciones de cada puesto.

## Instrucciones

### 1. Asignación de Puestos

Cada equipo recibirá ~3 puestos para documentar. La asignación se realizará en clase considerando:
- Disponibilidad de información (descripción de puesto y/o manual de funciones)
- Balance de complejidad entre equipos

### 2. Estructura de la Ficha de Puesto

Cada ficha debe incluir:

#### Información General
- Título del puesto
- Área/Departamento
- Nivel jerárquico (1-5)
- Reporta a:
- Supervisa a:
- Número de posiciones

#### Propósito del Puesto
- Resumen en 2-3 oraciones del objetivo principal del puesto

#### Responsabilidades Principales
- Lista de 5-8 responsabilidades clave
- Usar verbos de acción
- Ordenar por importancia/frecuencia

#### Requisitos del Puesto

| Factor | Especificación |
|--------|----------------|
| Educación mínima | |
| Educación deseable | |
| Experiencia mínima | |
| Certificaciones | |
| Idiomas | |
| Software/Herramientas | |

#### Competencias
- 3-5 competencias técnicas
- 3-5 competencias blandas

#### Condiciones de Trabajo
- Ubicación (oficina/campo/mixto)
- Horario
- Viajes requeridos
- Riesgos laborales

### 3. Puestos sin Descripción

Para los 7 puestos sin descripción de puesto disponible, deberán:
1. Analizar el manual de funciones
2. Investigar puestos similares en otras empresas
3. Proponer una descripción completa

## Entregables

1. **Fichas de puesto** (formato Word o PDF)
   - Una ficha por puesto asignado
   - Formato estandarizado

2. **Tabla resumen** comparando los puestos asignados en dimensiones clave

3. **Reflexión** (máx 300 palabras):
   - ¿Qué desafíos encontraron al estandarizar la información?
   - ¿Qué información adicional hubiera sido útil?

## Rúbrica de Evaluación

| Criterio | Excelente (10) | Bueno (8) | Suficiente (6) | Insuficiente (0-4) |
|----------|----------------|-----------|----------------|-------------------|
| Completitud | Todas las secciones completas con información relevante | Falta información menor | Secciones incompletas | Información insuficiente |
| Estandarización | Formato consistente, lenguaje profesional | Consistencia con errores menores | Formato inconsistente | Sin formato estándar |
| Investigación | Evidencia de investigación adicional para puestos sin descripción | Investigación básica | Investigación mínima | Solo copia de documentos |
| Reflexión | Análisis profundo de desafíos | Reflexión adecuada | Reflexión superficial | Sin reflexión |
